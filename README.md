# OranAI — Landing Page

Page de démonstration pour un assistant AI intelligent, avec chat interactif, style dark mode glassmorphism, et responsive design.

## Déploiement GitHub Pages

1. Crée un repo sur GitHub : `oranai-landing`
2. Clone-le :
```bash
git clone https://github.com/klasiinkov/oranai-landing.git
cd oranai-landing
```
3. Copie les fichiers de ce projet dans le dossier
4. Push :
```bash
git add .
git commit -m "Initial commit — OranAI landing page"
git push origin main
```
5. Va dans **Settings > Pages** et sélectionne `main` branch, dossier `/ (root)`
6. ✅ C'est en ligne !

## Structure

```
├── index.html      # Landing page avec chat demo
├── css/style.css   # Styles dark glassmorphism
├── js/main.js      # Animations & interactions
└── README.md       # Instructions
```

## Personnalisation

- Change le nom dans `index.html` (logo, titres, footer)
- Modifie les messages dans `.chat-messages`
- Couleurs dans `:root` du CSS
